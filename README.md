# Rock Paper Scissors

An implementation of [Rock Paper Scissors](https://en.wikipedia.org/wiki/Rock_paper_scissors) game in ReactJS. [Play the game](https://rock-paper-scissors-react-js.herokuapp.com/)

## Development stack
- Client: ReactJS
- Tools: Heroku
